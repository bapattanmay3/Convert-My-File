# Convert My File

"Convert My File" is a premium, high-fidelity file converter website that allows you to:
- **Convert**: PDF to DOCX, DOCX to PDF, and Images to PDF.
- **Translate**: Documents and text into 30+ languages.
- **Merge**: Combine multiple PDF files into one.
- **Compress**: Reduce the file size of PDFs and Images.

## Tech Stack
- **Backend**: Flask (Python)
- **Frontend**: HTML5, Vanilla CSS3 (Dark Theme)
- **Libraries**: `pdf2docx`, `docx2pdf`, `Pillow`, `deep-translator`, `PyPDF2`

## Local Setup
1. Clone the repository.
2. Create and activate a virtual environment.
3. Install dependencies: `pip install -r requirements.txt`.
4. Run the app: `python app.py`.
5. Open `http://127.0.0.1:5000` in your browser.
